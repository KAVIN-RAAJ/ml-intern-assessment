# Trigram Language Model

This directory contains the core assignment files for the Trigram Language Model.

## How to Run 

### Prerequisites
Ensure you have the required packages installed:
```bash
pip install -r ../requirements.txt
```

### Task 1: N-Gram Language Model
To train the model on *Alice's Adventures in Wonderland* and generate text:
```bash
python -m src.generate
```
To run the automated tests:
```bash
python -m pytest tests/test_ngram.py
```

### Task 2: Attention Mechanism
To run the demonstration script that verifies the attention implementation:
```bash
python -m src.attention_demo
```
